## 5.5. Representación Visual del Marco "Resistencia Encarnada" {#representacion-visual}

### Propósito de la representación visual

Un marco teórico complejo como el que hemos desarrollado se beneficia de una representación visual que sintetice su arquitectura y facilite su comprensión y aplicación. Esta sección describe la representación visual propuesta para el marco "Resistencia Encarnada", explica la lógica que informa sus decisiones de diseño, y anticipa su uso como herramienta de comunicación y análisis.

La figura que se describe a continuación deberá producirse posteriormente con herramientas de diseño gráfico apropiadas. Lo que ofrecemos aquí es una **descripción detallada** que sirva de especificación para esa producción, así como una explicación de por qué el marco se representa de esta manera particular.

### Descripción del diagrama propuesto {#diagrama-marco}

#### Estructura general: niveles concéntricos

La representación visual propuesta adopta una estructura de **niveles concéntricos** que refleja la arquitectura estratificada del marco:

```
┌─────────────────────────────────────────────────────────────────┐
│                    LENTE ÉTICO-POLÍTICA                         │
│  (atraviesa todos los niveles: cui bono, frontera, valor)       │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │           D4: CONTEXTUALIZACIÓN CULTURAL                │    │
│  │  (campo artístico, redes de cooperación, legitimación)  │    │
│  │  ┌─────────────────────────────────────────────────┐    │    │
│  │  │        D3: VALOR Y SIGNIFICADO ESTÉTICO         │    │    │
│  │  │  (intencionalidad, expresividad, aura, símbolo) │    │    │
│  │  │  ┌─────────────────────────────────────────┐    │    │    │
│  │  │  │     D2: TRABAJO CREATIVO-COLABORATIVO   │    │    │    │
│  │  │  │  (tareas, juicio situado, adaptación)   │    │    │    │
│  │  │  │  ┌─────────────────────────────────┐    │    │    │    │
│  │  │  │  │   D1: COGNICIÓN CORPOREIZADA    │    │    │    │    │
│  │  │  │  │   (Leib, conocimiento tácito,   │    │    │    │    │
│  │  │  │  │    enacción, sense-making)      │    │    │    │    │
│  │  │  │  └─────────────────────────────────┘    │    │    │    │
│  │  │  └─────────────────────────────────────────┘    │    │    │
│  │  └─────────────────────────────────────────────────┘    │    │
│  └─────────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────────┘
```

- El **núcleo central** (D1: Cognición Corporeizada) representa el fundamento epistémico: el cuerpo-vivido (*Leib*) como lugar del conocimiento.

- El **segundo anillo** (D2: Trabajo Creativo) muestra cómo la cognición se actualiza en sistemas laborales.

- El **tercer anillo** (D3: Valor Estético) representa la generación de significado artístico.

- El **cuarto anillo** (D4: Contextualización Cultural) sitúa todo en marcos socioculturales.

- El **marco exterior** representa la Lente Ético-Política, que no es un nivel adicional sino un atravesamiento que permea todos los niveles.

#### Lógica visual: del micro al macro

La estructura concéntrica comunica visualmente la lógica de niveles del marco:

- **Centro = micro**: El individuo cognoscente (el bailarín como *Leib*).
- **Anillos intermedios = meso**: Sistemas de trabajo y generación de valor.
- **Periferia = macro**: Contextos socioculturales e institucionales.

Esta progresión desde el centro hacia afuera refleja el flujo de fundamentación: cada nivel exterior presupone los interiores. No puede haber valor estético (D3) sin trabajo creativo (D2), ni trabajo sin cognición corporeizada (D1). Al mismo tiempo, los niveles exteriores retroalimentan los interiores, lo que se representa mediante **flechas bidireccionales** entre anillos.

#### Elementos visuales adicionales

**Flechas de interrelación**: Entre cada par de anillos adyacentes, flechas bidireccionales indican que la relación no es solo de fundamentación (hacia afuera) sino también de retroalimentación (hacia adentro).

**Textura de la lente ético-política**: El marco exterior (lente ético-política) se representa con una textura o patrón distintivo (por ejemplo, líneas diagonales) que indique que "atraviesa" todos los niveles. Alternativamente, puede representarse como un filtro semi-transparente superpuesto a toda la figura.

**Iconografía**: Cada dimensión puede incluir iconos representativos:
- D1: Silueta de cuerpo en movimiento
- D2: Engranajes o herramientas (trabajo)
- D3: Máscara teatral o símbolo estético
- D4: Edificio institucional o red de conexiones

**Palabras clave**: Dentro de cada anillo, las palabras clave principales de esa dimensión (como se indica en el diagrama ASCII anterior).

### Representación alternativa: estructura de pilares

Una representación complementaria podría adoptar la metáfora de **pilares que sostienen un edificio**:

```
                    ┌───────────────────────────────────┐
                    │      RESISTENCIA ENCARNADA        │
                    │   (Marco Teórico Integrado)       │
                    └───────────────────────────────────┘
                                    │
        ┌───────────────────────────┼───────────────────────────┐
        │                           │                           │
   ┌────┴────┐ ┌────────┐ ┌────────┴────────┐ ┌────────┐ ┌────┴────┐
   │   D1    │ │   D2   │ │       D3        │ │   D4   │ │ LENTE   │
   │Cognición│ │Trabajo │ │ Valor Estético  │ │Cultura │ │ ÉTICO-  │
   │Corpor.  │ │Creativo│ │                 │ │        │ │POLÍTICA │
   └─────────┘ └────────┘ └─────────────────┘ └────────┘ └─────────┘
        │           │              │               │           │
        └───────────┴──────────────┴───────────────┴───────────┘
                                   │
                    ┌──────────────┴──────────────┐
                    │   PRÁCTICA DANCÍSTICA       │
                    │   (Objeto de Análisis)      │
                    └─────────────────────────────┘
```

Esta representación enfatiza que las dimensiones funcionan como pilares que **sostienen** el marco completo, y que la lente ético-política atraviesa todos ellos como un elemento estructural transversal.

### Explicación de la lógica visual

#### Por qué niveles concéntricos

La estructura concéntrica se elige por varias razones:

1. **Comunica fundamentación**: El núcleo (D1) es literalmente el centro del que todo parte.

2. **Evita jerarquías valorativas**: A diferencia de una pirámide (donde la cúspide parece "superior"), los círculos concéntricos sugieren interdependencia más que jerarquía.

3. **Permite visualizar atravesamiento**: La lente ético-política puede representarse como permeando todos los niveles simultáneamente.

4. **Refleja la ontología del marco**: El cuerpo-vivido está en el centro, rodeado de capas sucesivas de contexto.

#### Por qué la lente como marco exterior

Representar la lente ético-política como marco exterior (no como un anillo más) comunica su función distintiva: no es una dimensión adicional que se suma a las otras, sino una **orientación crítica** que debe acompañar el análisis en cualquier nivel. Visualmente, esto se logra haciendo que el marco exterior "contenga" y "atraviese" los anillos interiores.

### Uso previsto de la representación visual

La figura propuesta servirá para múltiples propósitos:

1. **Síntesis conceptual**: Permitir a lectores captar rápidamente la arquitectura del marco.

2. **Herramienta pedagógica**: Facilitar la enseñanza del marco en contextos formativos.

3. **Guía de análisis**: Orientar la aplicación sistemática del marco a casos concretos.

4. **Comunicación interdisciplinaria**: Facilitar el diálogo con académicos y practicantes de otros campos.

### Nota para la producción gráfica

La figura definitiva deberá producirse con software de diseño gráfico que permita:

- Círculos concéntricos con colores o tonos distintivos para cada nivel
- Flechas bidireccionales entre niveles
- Textura o patrón para la lente ético-política
- Iconografía apropiada para cada dimensión
- Tipografía clara y legible
- Versiones en color y en escala de grises para diferentes contextos de publicación

Se recomienda que el diseño final sea revisado por especialistas en visualización de información para optimizar su eficacia comunicativa.

## Conclusión del Capítulo 5

El marco "Resistencia Encarnada" que hemos desarrollado a lo largo de este capítulo constituye la contribución teórica central de nuestra investigación. Integrando los fundamentos establecidos en los Capítulos 3 y 4, hemos construido un instrumento analítico y propositivo que permite:

1. **Articular con precisión** las dimensiones en que la práctica dancística presenta resistencias estructurales —no meramente técnicas— a la automatización.

2. **Evaluar configuraciones específicas** de tareas, prácticas y colaboraciones según su dependencia de las cualidades del *Leib*.

3. **Orientar el diseño** de colaboraciones humano-IA que respeten y potencien las cualidades irreducibles de la danza encarnada.

4. **Reflexionar críticamente** sobre las implicaciones políticas de las distinciones que el propio marco propone.

El marco no pretende cerrar el debate sobre la relación entre danza e inteligencia artificial; por el contrario, aspira a estructurarlo, proporcionando un lenguaje compartido y criterios operativos para una conversación que involucre a artistas, tecnólogos, educadores y responsables de políticas culturales.

El Capítulo 6 pondrá el marco a prueba mediante su aplicación a **viñetas analíticas** —casos de estudio conceptuales que demostrarán su poder heurístico y revelarán sus límites y posibilidades de refinamiento.
