# Prompt: Anexo C — Tabla Sintética de las Cuatro Dimensiones del Marco con Indicadores Operativos

## Contexto de la Tesis

**Título**: "Resistencia Encarnada: Hacia un Marco Teórico Interdisciplinario para el Análisis Crítico y la Co-Creación en la Danza ante la Inteligencia Artificial"

**Tipo de investigación**: Teórico-conceptual, constructivista, interdisciplinaria.

**Idioma de redacción**: Español académico. Todo el contenido debe escribirse en español.

**Marco teórico**: "Resistencia Encarnada" — un modelo de cuatro dimensiones analíticas (cognición corporeizada, trabajo creativo-colaborativo, valor y significado estético, contextualización cultural) atravesadas por una lente transversal ético-política. Cada dimensión opera en un nivel progresivo de complejidad: micro (individuo), organizacional (trabajo), intersubjetivo (valor estético), macro (sociocultural).

## Objetivo del Anexo

Producir una representación tabular completa y autocontenida de la arquitectura del marco "Resistencia Encarnada" que consolide en un instrumento de consulta rápida la información desplegada a lo largo de los capítulos 3, 4 y 5. Este anexo debe servir como referencia operativa para investigadores que deseen aplicar el marco sin necesidad de releer la tesis completa.

## Estructura del Anexo

### 2. Tabla Principal: Las Cuatro Dimensiones

Genera una tabla con las siguientes columnas para cada dimensión:

| Campo | Contenido requerido |
|-------|-------------------|
| **Dimensión** | Nombre completo y número (D1, D2, D3, D4) |
| **Nivel de análisis** | Micro / Organizacional / Intersubjetivo / Macro |
| **Definición formal** | Extraída directamente del Capítulo 5 (§5.2.1–5.2.4), resumida en 2-3 oraciones |
| **Tesis central** | La proposición principal que la dimensión sostiene (1 oración en negrita) |
| **Pilares teóricos** | Los autores/tradiciones que fundamentan la dimensión (con referencias: Merleau-Ponty, Varela, Autor et al., Benjamin, Becker, etc.) |
| **Elementos constitutivos** | Los 3-4 componentes que definen la dimensión (e.g., para D1: conocimiento tácito kinestésico, experiencia sensoriomotora de primera persona, inteligencia enactiva) |
| **Indicadores operativos** | Preguntas-guía concretas para evaluar una práctica en esta dimensión (3-5 preguntas) |
| **Ejemplo de resistencia alta** | Caso donde la dimensión muestra alta resistencia, extraído de las viñetas |
| **Ejemplo de resistencia baja** | Caso o sub-tarea donde la dimensión admite asistencia tecnológica |
| **Conexiones con otras dimensiones** | Cómo se relaciona con las demás (extraído de §5.4) |

**IMPORTANTE**: Dado que una tabla con tantas columnas sería ilegible, presenta cada dimensión como un **bloque tabular independiente** (4 tablas separadas, una por dimensión), con formato de dos columnas (Campo | Contenido).

## Criterios de Calidad

1. **Fidelidad**: Todas las definiciones, indicadores y ejemplos deben extraerse directamente de la tesis, no inventarse. Verificar cada afirmación contra los capítulos.
2. **Operatividad**: Las preguntas-guía deben ser lo suficientemente concretas para que un investigador pueda aplicarlas a un caso nuevo sin ambigüedad.
3. **Completitud**: Cubrir los cuatro componentes del marco + lente transversal sin omisiones.
4. **Legibilidad**: Las tablas deben ser claras y bien formateadas en markdown. No comprimir excesivamente.
5. **Autonomía**: El anexo debe ser comprensible sin releer la tesis, pero incluir referencias a secciones para quien quiera profundizar.
